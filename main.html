Welcome!

This is a borrowing system in which you can borrow and return things online.


Instructions

1. You can search through items you wish to borrow and view its ID and Details through the search bar.

2. You can borrow items through the borrow bar where you can input the details of the items you wish to borrow.

3. You can return borrowed items through the return bar where you input the details of the item you wish to return.
